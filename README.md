# Library Management System

## Project Description

The Library Management System is a Java Spring Boot web application developed to manage books, members, and book issue/return operations. It provides an easy-to-use interface for librarians to maintain records efficiently.

## Technologies Used

- Java 21
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap
- Maven
- Git & GitHub

## Modules

### Book Management
- Add Book
- Update Book
- Delete Book
- View Books
- Search Books

### Member Management
- Add Member
- Update Member
- Delete Member
- View Members

### Issue Management
- Issue Book
- Return Book
- View Issued Books

## Database

- books
- members
- issue_books

## How to Run

1. Clone the repository
2. Open in STS4
3. Configure MySQL
4. Run the Spring Boot application
5. Open http://localhost:8080
